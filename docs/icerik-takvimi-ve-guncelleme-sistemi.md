## 6. Eski İçerik Güncelleme Sistemi

3 ayda bir kontrol:

- [ ] Başlık güncel mi?

- [ ] İçerikte yeni proje örneği var mı?
- [ ] Yeni müşteri yorumu eklenebilir mi?
- [ ] İlgili hizmet sayfasına link var mı?
- [ ] İlgili portföy linki var mı?
- [ ] SSS bölümü genişletilebilir mi?
- [ ] Search Console’da gösterim alıp tıklama almıyor mu?
- [ ] Rakip içerikler daha güncel mi?

Güncelleme kararları:

- Gösterim var tıklama yoksa başlık/meta iyileştir.
- Ortalama konum 8-20 arasındaysa içerik genişlet.
- Eski yazıya yeni portföy ve yorum ekle.
- İç linkleri güncelle.
