# Google Görünürlük ve Dış Kaynak Playbook

Bu doküman Google Business Profile, Search Console, sitemap gönderimi, index isteği, backlink ve dış profil güncellemeleri için uygulanacak operasyon planıdır.

## 1. Google Business Profile

Durum: Kullanıcı hesabıyla kurulacak dış platform işi.

Yapılacaklar:

- [ ] Google Business Profile aç.
- [ ] İşletme adı: `NT Web Çözümleri`.
- [ ] Site bağlantısı: `https://ntwebcozumleri.com.tr`.
- [ ] Ana kategori: web tasarım / yazılım geliştirme / internet pazarlama kategorilerinden en uygun olanı seç.
- [ ] Hizmet bölgesi: Türkiye geneli / uzaktan çalışma vurgusu.
- [ ] Açıklama:

```text
NT Web Çözümleri; işletmelere özel web sitesi, admin panel, e-ticaret, ürün katalog, randevu, pazaryeri entegrasyonu, otomasyon ve özel yazılım sistemleri geliştiren butik yazılım stüdyosudur. Projelerde kapsamı baştan netleştirir, mümkün olduğunda canlı örneklerle ilerler ve teslim sonrası bakım/geliştirme desteği sunar.
```

Hizmetler:

- Kurumsal web sitesi geliştirme
- Admin panel geliştirme
- İşletme yönetim sistemi
- Pazaryeri XML/API entegrasyonu
- E-ticaret sitesi
- WhatsApp katalog sitesi
- Randevu sistemi
- Özel yazılım geliştirme
- Bakım ve geliştirme desteği

Google Business post planı:

- Haftada 1 blog yazısı paylaş.
- Her tamamlanan iş için kısa post yayınla.
- Her yeni yorum geldiğinde güven postu paylaş.
- Her yeni demo veya portföy eklendiğinde post yayınla.

## 2. Google Search Console

Durum: Kullanıcı Google hesabıyla kurulacak dış platform işi.

Kurulum:

- [ ] Domain property ekle: `ntwebcozumleri.com.tr`.
- [ ] DNS doğrulamasını domain panelinden yap.
- [ ] Alternatif olarak URL-prefix property ekle: `https://ntwebcozumleri.com.tr`.
- [ ] Sitemap gönder: `https://ntwebcozumleri.com.tr/sitemap.xml`.

İlk kontrol:

- [ ] Ana sayfa index durumu.
- [ ] `/services`
- [ ] `/projects`
- [ ] `/portfolio`
- [ ] `/blog`
- [ ] `/testimonials`
- [ ] `/pazaryeri-entegrasyonu`
- [ ] `/xml-entegrasyonu`
- [ ] `/admin-panel-gelistirme`
- [ ] `/stok-takip-sistemi`

Yeni içerik yayınlandıktan sonra:

- [ ] URL Denetleme aracına URL gir.
- [ ] Canlı URL test et.
- [ ] Index isteği gönder.

## 3. Haftalık Search Console Takibi

Her hafta kontrol edilecekler:

- [ ] Toplam tıklama
- [ ] Toplam gösterim
- [ ] Ortalama TO
- [ ] Ortalama konum
- [ ] En çok gösterim alan sorgular
- [ ] En çok tıklama alan sayfalar
- [ ] Gösterim alıp tıklama almayan sayfalar
- [ ] Indexlenmeyen sayfalar
- [ ] 404 hataları
- [ ] Mobil kullanılabilirlik sorunları
- [ ] Core Web Vitals uyarıları

Karar kuralları:

- Gösterim var, tıklama yoksa başlık ve meta açıklaması iyileştir.
- Konum 8-20 arasındaysa yazıya yeni bölüm, SSS ve iç link ekle.
- Konum 20+ ise konuya destekleyici blog veya landing page ekle.
- Blogdan teklif geliyorsa ilgili hizmet sayfasına CTA güçlendir.

## 4. Dış Profil ve Backlink Güncellemeleri

Güncellenecek profiller:

- [ ] Bionluk profilinden siteye link ver.
- [ ] LinkedIn profilinde site linki ekle.
- [ ] Instagram bio linkine site veya Bionluk ekle.
- [ ] GitHub profil README içine NT Web Çözümleri linki ekle.
- [ ] GitHub demo proje README dosyalarına canlı demo ve NT Web Çözümleri linki ekle.
- [ ] Google Business Profile’a site linki ekle.

Paylaşım yapılabilecek yerler:

- LinkedIn kişisel profil
- Instagram story/post
- GitHub README
- R10 / WM Aracı gibi platformlarda spam yapmadan uzmanlık içeriği
- Yerel işletme dizinleri
- Üniversite/proje bağlantılı güvenilir profiller

Paylaşım dili:

```text
Yeni bir rehber yayınladım: [Konu]
Küçük işletmelerin [problem] konusunda karar vermeden önce bilmesi gerekenleri sade şekilde anlattım.
Okumak için: [link]
```
