# Site Dışı Yapılacaklar

Bu dosya NT Works sitesinin koduna girmeyen, dış platformlarda yapılacak işleri tutar. Bunlar siteyi güçlendirir ama doğrudan repo içinde bitmiş sayılamaz.

## Google Search Console

- [x] Search Console kurulumu yapıldı.
- [ ] Sitemap gönderimi ve başarılı okundu kontrolü yapılacak.
- [ ] Ana sayfa, hizmetler, önemli landing sayfaları, portföy ve blog sayfaları URL inspection ile kontrol edilecek.
- [ ] Haftalık olarak 404, soft 404, 5xx, canonical, noindex, discovered/crawled not indexed raporları incelenecek.
- [ ] Gösterim alıp tıklama almayan sorgular not alınacak.
