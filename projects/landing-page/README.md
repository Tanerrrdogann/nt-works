# Landing Page Template

Tek hizmet, kampanya, urun lansmani veya on basvuru toplama sayfalari icin hazirlanmis premium conversion page sablonu.

## Amac

Bu template kurumsal site gibi firmayi uzun uzun anlatmaz. Ziyaretciyi tek hedefe goturur:

```text
Ziyaretci gelir -> hizmeti anlar -> guven duyar -> teklif / randevu / satin alma / WhatsApp aksiyonu alir
```

## Kullanim Alanlari

- Tek sayfalik satis sayfasi
- Reklamdan gelen kullaniciyi donusturme sayfasi
- Instagram / Bionluk / Google Ads yonlendirme sayfasi
- Tek hizmet tanitimi
- Kampanya sayfasi
- Urun veya hizmet basvuru sayfasi

## Ilk MVP Bolumleri

- Navbar / mini header
- Hero + ana teklif
- Trust bar
- Problem / solution
- Kimler icin uygun?
- Hizmet icerigi
- Surec
- Ornek sonuclar / demo alani
- Paketler
- Musteri yorumlari
- SSS
- Final CTA
- Footer
- Mobil sticky CTA bar
- WhatsApp linki
- Telefon linki
- Teklif formu gorunumu

## Ana Config Dosyasi

```text
src/config/site.ts
```

```text
brand
hero
trustItems
problemSolution
audience
offer
process
resultCards

testimonials

```

Kurumsal site firmayi genel olarak anlatir. Landing page tek hizmeti veya kampanyayi satar.

Her bolum su soruya cevap vermelidir:

```text
Bu kullanici neden simdi teklif alsin?
```
