# Source

Java source code for the `NT_os` virtual computer lives under `src/main/java`.

Current packages:

- `com.ntos.vm.bytecode`: opcode table
- `com.ntos.vm.loader`: `.ntbin` format reader/writer and loader
- `com.ntos.vm.memory`: `byte[]` virtual memory and memory layout
- `com.ntos.vm.os`: `NT_os` facade
